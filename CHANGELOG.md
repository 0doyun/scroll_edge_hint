## 0.1.0

- First release of `scroll_edge_hint`.
- Added `ScrollEdgeHint` and `ScrollEdgeHint.builder`.
- Added proportional leading/trailing strength model.
- Added `overlayFade` effect with custom `hintBuilder` support.
- Added example app with vertical/horizontal demos.
- Added widget tests for vertical, horizontal, reverse, and non-scrollable cases.
